HTML Element

The HTML tag defines the root of the HTML document. This is used in every HTML document.

Syntax

<html> TEXT </html>

Reference

w3schools.com

Header Element

The header tag defines information about the document.

Syntax

<head> TEXT </head>

Title Element

The title tag defines the title for the document.

Syntax

<title> TEXT </title>

Example

The title of this document is at the top of this page and says "HTML Reference Page."

The body tag defines the body of the document.

Syntax

Headings Element

The headings tag defines a header for a document of section. There are multiple headings that can be used.

Syntax

Example
