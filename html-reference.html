HTML Element

The HTML tag defines the root of the HTML document. This is used in every HTML document. The syntax of the tag is <html> TEXT </html>.

w3schools.com reference

Header Element

The header tag defines information about the document. The syntax of the tag is <head> TEXT </head>.

Title Element

The title tag defines the title for the document. The syntax of the tag is <title> TEXT </title>.

Example: the title of this document is at the top of this page and says "HTML Reference Page."

Body Element

The body tag defines the body of the document. The syntax of the tag is <body> TEXT </body>.
